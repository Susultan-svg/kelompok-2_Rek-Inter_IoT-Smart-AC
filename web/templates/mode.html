Smart AC Dashboard

Pilih Mode Kontrol

Otomatis

Raspberry Pi mengecek kamera tiap 10 detik. Jika ada orang AC dinyalakan bila masih mati; jika kosong 5 menit AC dimatikan.

Manual

Web menjadi kontroler AC. Raspberry Pi tetap membaca sensor dan kamera, tetapi tidak otomatis menyalakan atau mematikan AC.
